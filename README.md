Import the project in your IDE and run the class 'EventProducerApplication'.

An endpoint will be exposed at:

http://localhost:8080/events

which will return a json like:


"events": [
{
"name": "Event1",
"value": "Value1",
"id": 0
},
{
"name": "Event2",
"value": "Value2",
"id": 1
},
{
"name": "Event3",
"value": "Value3",
"id": 2
}
}
